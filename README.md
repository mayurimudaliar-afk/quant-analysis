# Quant Analysis Projects

Personal quantitative finance projects built during summer 2026 as part of preparation for Year 2 at NTU and recruiting for quantitative finance roles.

## Projects

### 1. Multi-Asset Intraday Tracker
**Files:** `multitracker.py`, `qqqm_tracker.py`

A real-time price and RSI monitoring tool for US-listed ETFs (VTI, QQQM) built using Python. Fetches live intraday data via yfinance and animates a candlestick chart with RSI panel, refreshing every 30 seconds.

**Tools:** Python, yfinance, matplotlib, FuncAnimation

**Concepts applied:** RSI calculation from scratch, moving averages, intraday candlestick visualisation, animated matplotlib figures

---

### 2. Microsoft Investment Analysis
**Folder:** `msft_analysis/`

An independent investment research project combining fundamental and technical analysis of Microsoft (MSFT). Designed to replicate the structure of a professional equity research note.

**Tools:** Python, yfinance, matplotlib, pandas

**Concepts applied:** Income statement analysis, FCF vs net income comparison, RSI signal generation, SMA crossovers

**Output:** Five charts saved as PNG, ready for a written investment report
